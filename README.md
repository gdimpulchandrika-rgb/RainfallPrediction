# Rainfall Prediction Web App 🌧️

This project predicts rainfall using a Machine Learning model and a Flask web application.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## Project Structure
RainfallPrediction
│
├── static
├── templates
├── app.py
├── main.py

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python main.py

3. Open in browser

http://127.0.0.1:5000/
